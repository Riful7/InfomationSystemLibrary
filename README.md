# InfomationSystemLibrary
Репозиторий с готовым проектом информационной системы управления библиотекой для учебной (технологической) практики

# Содержание проекта
* uis - папка с файлами интерфейса приложения формата .ui и .py
* test_data - папка с тестовыми данными формата .xlsx
* sql - папка с бэкапом базы данных "InfoSysLibrary"

# Используемые средства
* Python 3.12.1
* Pandas 3.0.3
* PySide6 6.11.1
* PostgreSQL 17.6
